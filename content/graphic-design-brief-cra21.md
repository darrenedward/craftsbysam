### Quality Standards

**Resolution and Quality:**
- Print templates: 300 DPI, CMYK color mode
- Web/social: 72 DPI minimum, RGB color mode
- Create all assets at 2× final size for crispness
- Use anti-aliasing on all text and shapes
- No compression artifacts on PNG exports

**Accessibility:**
- Text contrast minimum 4.5:1 (WCAG AA)
- Avoid using color as the only way to convey information
- Provide alt text considerations for images
- Ensure text is readable at small sizes

**Consistency:**
- Use exact color values from Brand Style Guide
- Maintain typography hierarchy consistently
- Logo placement should follow established patterns
- Spacing and alignment must be precise

### Technical Requirements

**Software and Formats:**
- **Editable Source Files:** SVG (preferred) or PSD/AI
- **Final Delivery:**
  - DOCX for business card and letterhead (Microsoft Word compatible)
  - PNG for all web/social assets (transparent backgrounds where applicable)
  - PDF for print templates (press-ready)
- **Color Profiles:**
  - Print: CMYK (FOGRA39 or GRACoL standard)
  - Web: sRGB

**Export Checklist:**
- [ ] All text converted to outlines or fonts included/embedded (for SVG/PSD)
- [ ] Transparent backgrounds on web assets where required
- [ ] Proper bleed on print items (3mm)
- [ ] Safe zones observed (no critical elements near edges)
- [ ] File sizes optimized (PNG compression balanced, no quality loss)
- [ ] Multiple format variations provided where needed

## Brand Consistency Rules

**Non-Negotiable Brand Elements:**
1. **Color Palette:** Only use colors from official Brand Style Guide
2. **Typography:** Lora for headlines, Open Sans for body text — no substitutions
3. **Logo:** Use provided SVG logos only — do not recreate or modify
4. **Voice:** All copy/text should feel warm and personal (even placeholders)

**Creative Flexibility:**
- Layout and composition are your creative space
- Decorative elements should complement, not compete
- Photography style should be natural and warm
- Feel free to suggest improvements to this brief's layout recommendations

**What to Avoid:**
- Deprecated pink/plum color palette from earlier drafts
- Pure black (#000000) — always use Charcoal (#3D3D3D)
- Overly corporate or formal design language
- Cluttered layouts — keep it simple and focused
- Excessive emoji use (max 2-4 per social design, less for print)
